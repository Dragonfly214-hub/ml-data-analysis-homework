# 内容简介
在大作业中，我完成了一个数据分析的流程，包括有：
对数据进行查看和可视化  
用课上教过的机器学习模型进行训练  
针对最后的模型预测结果进行分析和对比  
对实验结果的合理性进行验证  
# 使用的数据集
使用的数据集是California Housing Dataset，来源于scikit-learn 内置数据集，选择了一个比较好实现的预测房价的回归任务
# 我在其中使用的方法
线性回归，随机森林回归
由于改起来比较复杂，所以模型参数就使用了它默认给的设置
# 评价方式
整个的模型达到的效果主要通过以下指标进行评估：
RMSE，MAE，R²  
# 文件说明
第一个文件：对数据进行查看以及一些基本的分析  
第二个文件：对模型进行训练以及针对不同模型的结果对比分析，给出我自己分析的结论  
第三个文件：产生了预测的结果，然后对此进行分析验证
